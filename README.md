# AMD Ryzen Master 繁體中文翻譯

以官方簡體中文進行用詞修正，翻譯並不完美

用法：將 Languages 覆蓋至 `C:\Program Files\AMD\RyzenMaster\bin` 重啟 Ryzen Master 即可

*已於 2.10.2.2367 上測試
